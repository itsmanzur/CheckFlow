Fast one-page checkout foundation for WooCommerce with admin UX, i18n, and dynamic feature roadmap support.

== Description ==

CheckFlow is a WooCommerce-focused checkout plugin foundation. This version includes:
- Admin panel shell with scoped styles
- Multi-language admin strings with overrides
- Architecture scaffold for checkout, analytics, tracking, courier, and payment modules

== Installation ==

1. Upload plugin folder to `/wp-content/plugins/`.
2. Activate `CheckFlow`.
3. Ensure WooCommerce is active.
4. Open `CheckFlow` from WordPress admin menu.
